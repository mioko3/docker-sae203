Présentation des TP Docker

TP 1 : Introduction à Docker et Conteneurs

  • Installation et Configuration : Installation de Docker et vérification.
  • Exécution d’un Conteneur : Télécharger une image et lancer un conteneur.
  • Gestion des Conteneurs : Lister, arrêter et supprimer des conteneurs.
  • Volumes et Réseaux : Créer et gérer des volumes et des réseaux.

TP 2 : Création d’Images avec Dockerfile

  • Créer un Dockerfile : Personnalisation d’une image avec un Dockerfile.
  • Construire l’Image : Créer une image à partir du Dockerfile.
  • Lancer un Conteneur : Exécuter un conteneur à partir de l’image.
  • Installer des Services : Ajouter Apache à une image Debian.
  • Déploiement via GitHub : Sauvegarder et déployer via GitHub.
  • Déploiement d’un Serveur LAMP : Déployer Apache, MariaDB, PHP avec Docker.

Présentation du projet

Projet SAE 2.03

  • Objectif : installer un service réseau avec Docker.
  • Base : image Docker debian:latest obligatoire.
  • Choix libre : chaque équipe choisit un service (GitLab, LDAP, VOD...).
  • Limitation : un seul Dockerfile, pas de Docker Compose.
  • Hébergement : projet sur GitHub (dépôt docker-sae203).
